## What is templ-icons-lucide ?

It is a go package used to provide icons for the Go templ components.
This package makes it easy to integrate Lucide icons into your Go templ components.
Lucide is a popular, open-source set of high-quality SVG icons that are widely used for web development and UI design.
With templ-icons-lucide, you can easily include these icons within your Go templ components without needing to manage SVG files directly.

## Key Features

- **Predefined Lucide Icons**: Access a wide range of icons from the Lucide icon set, directly within your Go temp components.
- **Easy Integration**: Incorporating icons into your Go web applications is as simple as adding a templ function call. No need to deal with external SVG files.
- **Flexible Customization**: Customize icon size, color, and other attributes easily using Goâ€™s templ component functions.
- **Consistent Design**: Lucide provides a cohesive and visually appealing set of icons that enhance the overall user experience.

Whether you're building a full-fledged web application or a small project,
`templ-icons-lucide` offers a straightforward way to bring icons into your Go-powered templ components.

## Supported Icons

This package includes a variety of icons from the **Lucide** icon set.
For the full list of available icons and their usage, refer to the [Lucide icon set](https://lucide.dev/) documentation.
